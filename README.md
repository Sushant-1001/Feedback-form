# ⭐ Feedback Management System (Flask + MySQL)

A simple web application built using Flask that allows users to submit feedback and view stored feedback from a MySQL database.

---

## 🚀 Features

- Submit feedback form
- Store data in MySQL database
- View all feedback entries
- Delete all feedback records
- Clean and responsive UI

---

## 🛠️ Tech Stack

- Python (Flask)
- HTML5
- CSS3
- MySQL
- Flask-MySQL Connector
